# Keyboard

The on-screen keyboard lets you type text for searches, dictionary lookups, and
other input fields.

## Basics

- **Lock modifier keys** вҖ” Tap _ALT_ or _SHIFT_ twice to lock them on.
- **Combine key (CMB)** вҖ” Press _CMB_ followed by two keys to type accented or
  special characters (see [Combination Sequences](#combination-sequences)
  below). For example, `CMB o e` produces _Е“_.
- **Delete / motion keys** вҖ” Tap and hold to act on whole words instead of
  individual characters.
- **Keyboard layouts** вҖ” Tap and hold the space bar to switch layouts.

## Inputting Accented Characters

Cadmus supports these through **combination sequences** using the
_CMB_ key. See [Combination Sequences](#combination-sequences).

### How to type an accented character

1. Press the **CMB** key.
2. Press the **base letter** (e.g. `e`).
3. Press the **accent modifier** (e.g. `'` for acute).

Example: `CMB e '` produces **Г©**.

## Custom Keyboard Layouts

Keyboard layouts are defined as JSON files with the following structure:

- **name** вҖ” Display name shown in the keyboard layouts menu.
- **outputs** вҖ” List of output keys for each modifier combination (_none_,
  _shift_, _alt_, _shift+alt_).
- **keys** вҖ” Description of each key. Special key names:
  _Shift_ (_Sft_), _Return_ (_Ret_), _Alternate_ (_Alt_), _Combine_ (_Cmb_),
  _MoveFwd_ (_MoveF_, _MF_), _MoveBwd_ (_MoveB_, _MB_), _DelFwd_ (_DelF_, _DF_),
  _DelBwd_ (_DelB_, _DB_), _Space_ (_Spc_). Use _в–ў_ to mark output keys.
- **widths** вҖ” Width/height ratio for each key. The key gap ratio is 0.06.
